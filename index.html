Why `SOUL.md` is central

Culture drives everything else.

Memory only matters if it is carried with respect. People only participate if they feel safe and welcome. Programme only works when it reflects inclusive values in practice.

Design logic

One core, three active links.

The graphical view makes the markdown structure legible at a glance, so the homepage communicates both event identity and content architecture in the same frame.
